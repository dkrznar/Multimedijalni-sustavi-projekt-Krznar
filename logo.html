Izrada logotipa u Inkscape-u

U nastavku je prikazan logotip i postupak izrade u programu Inkscape.

Postupak izrade

  1. Kao polazište za izradu logotipa koristila sam javno dostupan vektor kuće preuzet sa stranice FREE SVG. Vektor kuće dodala sam u novi dokument opcijom File → Import.
  2. Budući da je kuća bila sastavljena od više zasebnih elemenata, najprije sam odabrala sloj na kojem je bio element koji je predstavljao dimnjak te ga uklonila tipkom Delete.
  3. Također, budući da je preuzeti vektor imao vrata, a smatrala sam da je za pozadinu iza teksta vizualno primjereniji prikaz kuće bez vrata, pomoću alata Rectangle Tool crnim pravokutnikom prekrila sam dio koji predstavlja vrata.
  4. Držeći tipku Shift, odabrala sam novokreirani pravokutnik, element koji predstavlja kuću i element koji predstavlja krov te sam ih opcijom Path → Union ujedinila u jedan sloj koji sam radi lakšeg snalaženja preimenovala u “Kuća”.
  5. U sljedećem koraku, dodala sam tekst “BoPiCa” koristeći Text Tool. Odabrala sam željenu veličinu fonta, nakon čega sam u panelu Fill and Stroke s desne strane ekrana odabrala boju teksta - Fill, boju obruba - Stroke paint te željenu debljinu obruba (1.565 mm) i izgled obruba (Join: Round join; Cap: Round cap; Order: Stroke, Fill, Markers) - Stroke style.
  6. Pomoću alata Pen Tool napravila sam zaobljenu liniju te sam označavanjem sloja teksta i sloja linije koristila opciju Text → Put on Path kako bih zaoblila tekst prema liniji. Koristeći Selector Tool pomaknula sam tekst na željeno mjesto na logotipu.
  7. Osim teksta “BoPiCa” opcijom za dodavanje teksta dodala sam tekst “SKLONIŠTE”. Sloj teksta “SKLONIŠTE” i sloj kuće spojila sam u jedno opcijom Path → Difference.
  8. Završna faza izrade logotipa obuhvaćala je dodavanje elementa šapice iznad teksta. Vektor šapice preuzela sam sa stranice SVG Repo te sam ga, kao i vektor kuće, dodala opcijom File → Import.
  9. Vektor šapice sastojao se od više elemenata pa sam sve elemente, radi jednostavnije manipulacije slojem, označila držeći tipku Shift te ih objedinila u jedan sloj pomoću opcije Path → Union.
  10. Boju šapice promijenila sam u željenu boju na panelu Fill and Stroke → Fill. Korištenjem alata Ellipse/Arc Tool, šapicu sam prekrila krugom na koji sam primijenila crno-bijeli Radial gradient.
  11. Taj efekt primijenila sam na šapicu korištenjem maske. Najprije sam označila sloj šapice, zatim sloj kruga te koristila opciju Object → Mask → Set Mask. KoristećiSelector Tool, šapicu sam pozicionirala na željeno mjesto.
  12. Gotov logotip izvezla sam u .png formatu putem opcije File → Export.
