# Multimedijalni sustavi 
Projekt za predmet Multimedijalni sustavi. U projektu je bilo potrebno prezentirati korištenje programa za obradu slika, videa i zvuka. 
U projektu je prikaznano korištenje programa Inkscape, GIMP, DaVinci Resolve, Affinity i Audacity.
